### **HELM: Hyper(graph) Experiment and Learning Manager**

Author: Sharjeel Mustafa

This repository provides a generic pipeline to easily benchmark various hypergraph and graph machine learning models. Given that hypergraphs generalize graphs the pipeline naturally operates on graphs as well. It does not provide hypergraph specific tool, only handling the executation phase.

Present Limitations:
1. Transductive-tasks only
2. Node classification
